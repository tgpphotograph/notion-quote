# 每日金句 Widget

> **全螢幕、繁體中文、點擊即換、每天自動更新**

一個輕量級的每日金句小工具，支援嵌入 Notion、Obsidian、任何支援 iframe 的平台。

## ✨ 特色

- ✅ **繁體中文**：用彩虹卡的概念，寫出一句句正能量語錄
- ✅ **每天一句**：localStorage 快取保證同一天固定一句，隔天自動更新
- ✅ **點擊換句**：全螢幕點擊隨時換新句，淡入動畫
- ✅ **全螢幕自適應**：完美填滿容器，文字大小自動調整
- ✅ **離線可用**：快取後無網路也能顯示
- ✅ **客製化**：可調整金句類型、配色、字體

## 📱 嵌入方式

### Notion
1.貼入網址 https://memocubee.github.io/DailyZen/

2.點嵌入

## 🎛️ 客製化選項

哲理: ?c=k

勵志: ?c=i  

詩詞: ?c=h

文學: ?c=b

動漫: ?c=d

混合範例：?c=k&c=i&c=h&c=b

## 🎨 配色主題

修改 CSS 的 background、color：

藍色系: #e8f4f8 → #a8d5e2

深色系: #1a1a1a → #2d2d2d
